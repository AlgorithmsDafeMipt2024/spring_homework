# Задача о размене монет

Дано число и вектор с номиналами монет, нужно вернуть минимальное количество монет которое можно набрать данными номиналами.

Примеры:
1) Дано число 14, и номиналы [1, 2, 5, 10], мы должны вернуть число 3 (10 * 1 + 2 * 2)
2) Дано число 19, и номиналы [1, 2, 5, 10], мы должны вернуть число 4 (10 * 1 + 5 * 1 + 2 * 2)
